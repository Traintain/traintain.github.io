Experience

Integration of python plugin in a web-based medial-image viewer

The Alberta Stroke Program Early CT score (ASPECTS) is an score used by neuroradiologists to estimate the extent of the damage in a patient with an ischemic stroke.

An automated ASPECTS algorithm was developed in python and deployed using AWS as part of the Appremia project. In order to visualize the segmentations we connected the algorithm output into Atix, a web-based medical image viewer.
